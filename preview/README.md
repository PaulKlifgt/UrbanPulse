# Preview

Папка содержит презентационную страницу для Streamlit-маршрута `/preview`.

Чтобы заменить демонстрационные материалы на реальные:

- положите скрин главной панели как `preview/assets/dashboard.png`
- положите скрин детального района как `preview/assets/district.png`
- положите скрин блока рынка/архитектуры как `preview/assets/market.png`
- положите видео-демо как `preview/assets/demo.mp4`

Если эти файлы не добавлены, страница использует локальные SVG-плейсхолдеры и текстовый блок для видео.
